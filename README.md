# FinTale

FinTale adalah aplikasi manajemen keuangan pribadi _(personal finance)_ berbasis mobile yang dikembangkan menggunakan Flutter. Berbeda dengan aplikasi keuangan biasanya, FinTale membawa pendekatan Gamifikasi.

Aplikasi ini dirancang khusus untuk membantu pengguna melunasi hutang, mengumpulkan dana darurat, dan membangun portofolio investasi dengan cara yang menyenangkan layaknya bermain Role-Playing Game, namun tetap dibalut dalam desain UI yang bersih, modern, dan profesional bergaya E-Wallet Premium.

## 🌟 Konsep Unik

FinTale memiliki fitur Dual-Mode Terminology. Pengguna dapat mengubah antarmuka aplikasi secara instan antara istilah keuangan standar dan istilah game RPG.

- Total Saldo <--> Total HP (Darah)
- Alokasi Biaya Hidup <--> Mana / Potion
- Daftar Hutang <--> Boss Raid
- Tagihan Rutin Bulanan <--> Quest Bulanan
- Pemasukan / Gaji <--> Loot / Drop
- Investasi Saham / Crypto <--> Fighter / Assassin
